## Awal mula

Saya mulai belajar Next.js karena ingin punya blog sendiri tanpa bergantung pada platform orang lain.

## Apa yang dipelajari hari pertama

Hal pertama yang membingungkan adalah perbedaan **Server Component** dan **Client Component**.
Server Component dirender di server — tidak ada JavaScript yang dikirim ke browser.
Client Component dirender di browser — bisa punya state dan event handler.

## Kesimpulan

Next.js ternyata tidak seseram yang dibayangkan. Mulai dari project kecil adalah keputusan terbaik.
